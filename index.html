Instructions

  • 1. Firstly, the game shows the first colour in the sequence (eg. blue).
  • 2. The user clicks on the blue button.
  • 3. Next, the game shows the next colour (red), the user has to remember the sequence : that is blue, red and so on and so forth.
  • 4. If the user messes up the sequence, then the game ends.
